# Demo Day Alpha — Evaluation Framework

## Scoring Matrix

Each startup is evaluated on 6 dimensions, weighted as follows:

| Dimension | Weight | What We're Assessing |
|-----------|--------|---------------------|
| Team | 25% | Founder quality, complementarity, domain expertise, coachability |
| Market | 20% | TAM/SAM credibility, growth dynamics, regulatory tailwinds/headwinds |
| Product | 20% | Defensibility, production-readiness, user feedback, technical debt |
| Traction | 20% | Revenue, pipeline, retention, growth rate |
| Strategic Fit | 15% | Alignment with Vertex Minds thesis, value-add potential, co-investor appeal |

## Scale

Each dimension scored 1–5:

- **5** — Exceptional. Top decile of what we see at this stage.
- **4** — Strong. Clear evidence, few concerns.
- **3** — Adequate. Meets expectations for stage, some gaps.
- **2** — Below expectations. Significant concerns that need addressing.
- **1** — Disqualifying. Fundamental problems.

## Thresholds

- **Weighted average ≥ 4.0**: Strong INVEST candidate. Proceed to term sheet unless specific red flag.
- **Weighted average 3.2–3.9**: Discussion zone. IC deliberation required.
- **Weighted average < 3.2**: PASS unless compelling qualitative argument.

No startup with any single dimension scored 1 will receive an INVEST recommendation, regardless of weighted average.

## Process

1. Each IC member (Daniel, Ava) scores independently during the pitch + Q&A
2. Tomás records scores and flags divergences >1.5 points on any dimension
3. Post-event deliberation: IC discusses each startup, reconciles scores, reaches consensus
4. Final classification: INVEST / CONDITIONAL / PASS
