# Caso B — FireGlass Platform Development

**Domain:** IoT / Cloud Platform / Product Development

## Scenario

Innovative Windows LLC (40 employees, Verona, Italy) manufactures smart fire-resistant window systems with embedded IoT sensors. They are building FireGlass, a cloud-based management platform to monitor installations, manage field operations, and enable recurring software revenue as they scale into European and North American markets.

The project involves MQTT protocol integration, firmware coordination with hardware partner Helion Dynamics, field testing, and sprint-based software development.

## Epistemic Characteristics

This corpus is designed to contain:

- **Technical decisions** (architecture choices, protocol selection) with supporting rationale
- **Evolving plans** across sprint cycles with scope adjustments
- **External dependencies** (Helion firmware) creating blocking constraints
- **Field testing observations** that may contradict initial assumptions
- **Budget and timeline concerns** that escalate through the project
